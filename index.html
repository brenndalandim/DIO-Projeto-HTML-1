Introdução ao HTML na Prática

(voltar)

Ferramentas e informações para iniciar seu estudo sobre HTML, aprenda sobre a estrutura de um HTML, o funcionamento de tags, textos e listas para criar sua primeira página do ZERO

Ferramentas Utilizadas

(voltar)

Nesse curso iremos usar o Microsoft Visual Code para criar nossa página, um editor de código-fonte autônomo que é executado no Windows, macOS e Linux. A melhor escolha para desenvolvedores JavaScript e Web, com toneladas de extensões dar suporte a praticamente qualquer linguagem de programação.

Utilizaremos também a extensão Live Server, que pode ser encontrada na aba de extensões do VS Code, para que nossa página seja acessível e visualizada pelo navegador.

Inspetor de Elementos

(voltar)

Uma ferramenta do navegador para analisar e entender o funcionamento da página através de seu código HTML.

Através dele conseguiremos ver tipos de formatação, comportamento padrão, modificações da página, etc.

Para acessá-lo basta clicar com o botão direito na página e selecionar Inpecionar Elemento.

Estrutura Básica do HTML

(voltar)

Para iniciar é necessário selecionar uma pasta no VS Code que será sua área de trabalho e criar um arquivo index.html (nome padrão por convenção para a página inicial)

Todo o código html é baseado em tags (<>), elas que dizem o formato da página, elas podem ser tags únicas (<>) ou tags que precisam ser fechadas (<></>).

Uma página se inicia com a tag html <html></html>, dentro dela que iremos escrever nosso código.

Dentro desse espaço temos as tags head e body que são a parte não visível e a parte visível de um site.

Na tag head <head></head> colocamos informações de configurações não visíveis da página como seu título, idioma, etc.

=> definimos o título com a tag <title> nome da página </title>, ele aparecerá na aba da página

Já na tag body <body></body> configuramos o conteúdo da página, toda sua parte visível.

=> escrever um texto qualquer dentro dessa tag aparecerá na tela

Com o plugin do live server instalado, conseguimos colocar esses códigos no index e visulizar a página clicando com o botão direito no arquivo e depois "open with live server", isso abrirá o navegador com a página carregada, sempre que uma alteração for feita ela já é refletida nessa página.

Falando sobre Tags

(voltar)

Como vimos anteriormente, as tags são o que formatam a página como desejamos e elas podem ser únicas ou com fechamento

Uma dica para não confundir o tipo da tag é: quando a tag é responsável por alterar somente uma parte específica do código ela tem fechamento, quando só tem uma funcionalidade não tem fechamento

=> a tag img não tem fechamento porque só adiciona uma imagem

=> a tag strong tem porque modifica somente um pedaço do código

Veremos sobre essas tags em breve, por enquanto ficamos apenas com o entendimento

Atributos das Tags

(voltar)

Atributos são configurações para alterar a forma padrão de uma tag, algumas tem específicos e outras possuem atributos globais (o mesmo atributo funcionando em tags diferentes).

Um exemplo de atributo é o id="" que designa um nome único para uma tag para conseguir configurá-la através desse nome, sendo ele seu identificador. Esse recurso é mais utilizado posteriormente no JavaScript e CSS.

=> <img id="nome_do_identificador">

Outro exemplo é a o atributo style="" que estiliza o conteúdo da tag.

=> <p style="color: blue"> - transforma o texto em azul

Por último temos o atributo class="" que define um identificador que pode ser usado em várias tags para aplicar uma mesma configuração ou estilo em várias tags diferentes.

=> <img class="nome_do_identificador">

Esses foram alguns exemplos de atributos globais, mas existem os específicos como o type="" que funciona na tag input para definir que tipo de caixa irá aparecer na tela, o src="" usado na tag img para dizer o caminho da imagem desejada, o href="" usado na tag a para referenciar o link desejado, entre outras.

Textos

(voltar)

Algumas tags de formatação de texto:

  • <strong> teste </strong> = teste - texto em negrito
  • <i> teste </i> = teste - texto em itálico
  • <u> teste </u> = teste - texto sublinhado
  • <mark> teste </mark> = teste - texto com marca texto
  • <sup> teste </sup> = teste - texto sobrescrito
  • <small> teste </small> = teste - texto pequeno
  • <q> teste </q> = teste - formata uma citação
  • <h1> teste </h1> - utilizando do h1 ao h6 são criados títulos
  • <blockquote> teste </blockquote> - formata uma citação num bloco separado
  • <hr> - cria uma linha separadora
  • <input type="text"> = - cria uma caixa de texto
  • => existem outros tipos de input, teste trocar a palavra "text" por number, color e outras opções disponíveis e descubra novas ferramentas HTML.
  • * dica: quando você digita no VS Code aparecem algumas possíveis formas de completar seu código, use isso para ver as possibilidades e aprender coisas novas na prática. Faça o teste acima selecionando as opções que o editor te da!

Listas Ordenadas e Não Ordenadas

(voltar)

Conseguimos criar listas com as tags <ol> e <ul>

A tag <ol> cria listas ordenadas por número, letras ou outros símbolos que denotem uma ordem

  1. teste
  2. teste
  3. teste

A tag <ul> cria listas não ordenadas com símbolos que não denotem uma ordem

  • teste
  • teste
  • teste

Para adicionar elementos as listas usamos a tag <li>

=> <ol> <li> teste </li> </ol>

Para estilizar as listas com os símbolos desejados usamos a tag style="" dentro das tags <ol> ou <ul> se quiser mudar todos os símbolos da lista ou dentro da tag <li> caso queira mudar somente um símbolo específico.

Também é possível adicionar listas dentro de listas, sendo um elemento <li> da lista, uma nova lista com a tag <ul> ou <ol>.

Links

(voltar)

Links são criados com a tag <a>, nela iremos usar o atributo href="" para informar o link que será utilizado.

=> <a href="#"> clique aqui </a> = clique aqui - no lugar do # colocamos o link desejado

Nesse atributo de referência podemos colocar também referências ao próprio site, como, por exemplo, queremos ser levados a outra página, criamos outro arquivo html para criar uma nova página (about.html) e colocamos o nome dele como referência.

=> <a href="about.html"> clique aqui </a> = clique aqui - isso levaria a página "sobre" do site

Também é possível referenciar partes da mesma página utilizando o atributo id="", nomeando um identificador a alguma parte do conteúdo da página e depois referenciando ele para ser levado para lá.

=> <a href="#nome_do_identificador"> clique aqui </a> = clique aqui - isso levaria a parte da página com o identificador

Temos ainda o atributo target="" que diz se, ao clicar no link, ele será aberto em uma nova aba (_blank) ou na sua aba atual (_self).

=> <a href="" target="_blank"> clique aqui </a> = clique aqui - uma nova aba é aberta após clicar no link

Quando não usamos o atributo target="" a configuração padrão é que o link seja aberto na aba atual, não sendo necessário usar o _self.

Além disso, o atributo title="" cria um balão com um texto que aparecerá quando o mouse passar em cima do link

=> <a href="" target="_blank" title="texto"> clique aqui </a> = clique aqui - aparecerá um balão escrito "texto"
